# Smart Locker IDS/IPS

Дипломный проект на тему:

«Разработка и внедрение программного обеспечения IDS/IPS для предприятия в сфере обслуживания»

---

# Описание проекта

Smart Locker IDS/IPS — программный комплекс для мониторинга сетевого трафика, обнаружения сетевых атак и автоматической блокировки подозрительной активности.

Система разработана на языке Python и функционирует в среде Astra Linux.

---

# Основные возможности

- мониторинг сетевого трафика в режиме реального времени;
- анализ пакетов IP, TCP, UDP, ICMP;
- обнаружение SYN Flood атак;
- обнаружение сканирования портов (Port Scanning);
- автоматическая блокировка IP-адресов через iptables;
- поддержка whitelist (белого списка);
- журналирование событий безопасности;

---

# Структура проекта

main.py           Точка входа в систему
monitor.py        Мониторинг и перехват трафика
ids.py            Логика обнаружения атак
ips.py            Блокировка IP-адресов
logger.py         Журналирование событий
config.py         Конфигурация системы

---

# Требования

Для работы системы необходимы:

- Astra Linux / Debian-based Linux
- Python 3.10+
- установленный iptables

---

# Настройка

Настройки системы находятся в файле:

```bash
config.py
```

Доступные параметры:

```python
INTERFACE = "eth0"

SYN_THRESHOLD = 10
PORT_SCAN_THRESHOLD = 20

ENABLE_IPS = True

WHITELIST = [
    "127.0.0.1",
    "192.168.1.1"
]
```

## Описание параметров

- `INTERFACE` — сетевой интерфейс для мониторинга;
- `SYN_THRESHOLD` — порог SYN-пакетов;
- `PORT_SCAN_THRESHOLD` — порог сканирования портов;
- `ENABLE_IPS` — включение/отключение IPS;
- `WHITELIST` — список доверенных IP-адресов.

---

# Запуск системы

Запуск производится с правами администратора:

```bash
sudo python3 main.py
```

После запуска система начнет мониторинг сетевого трафика.

---

# Принцип работы

1. Модуль `monitor.py` перехватывает сетевые пакеты через Scapy.
2. Информация о пакетах передается в IDS-модуль.
3. `ids.py` анализирует трафик:
   - SYN Flood;
   - Port Scanning.
4. При обнаружении атаки вызывается IPS-модуль.
5. `ips.py` блокирует IP через iptables.
6. События записываются в журнал.

---

# Пример блокировки IP

Система автоматически выполняет команду:

```bash
iptables -A INPUT -s <IP> -j DROP
```
